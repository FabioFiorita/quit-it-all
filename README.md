# Quit It All

A lightweight macOS menu bar app to close all running applications with one click.

## Download

Get the latest release from the [Releases page](https://github.com/FabioFiorita/quit-it-all/releases).

## Features

- **Quit All Apps** — gracefully closes all running apps (except Finder)
- **Force Quit All Apps** — force-kills unresponsive apps
- Lives in the menu bar — no Dock icon clutter
- Shows a list of currently running apps
- Signed and notarized for macOS Gatekeeper

## Build from source

Requires Xcode and Swift 5.9+.

```bash
./build.sh
open QuitItAll.app
```

## Requirements

- macOS 13 (Ventura) or later
